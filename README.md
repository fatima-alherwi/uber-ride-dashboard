**Uber Ride Analytics Dashboard
Project Overview**

This project presents an interactive Power BI dashboard analysing Uber ride operations. The dashboard provides insights into bookings, revenue, ride performance, cancellations, payment methods, and ride locations.

**Tools Used**
- Microsoft Power BI
- Power Query
- DAX
- Data Modelling

**Dashboard Pages**

**1. Overview**
- Total Bookings
- Completed Rides
- Cancelled Rides
- Revenue
- Customer Count
- Booking Trends

**2. Ride Performance**
- Average VTAT
- Average CTAT
- Customer Rating
- Driver Rating
- Revenue by Vehicle Type
- Ride Distance Trends

**3. Cancellation Analysis**
- Customer Cancellations
- Driver Cancellations
- Incomplete Rides
- Cancellation Reasons

**4. Payment & Location Analysis**
- Payment Methods
- Revenue by Payment
- Top Pickup Locations
- Top Drop Locations

**Key Insights**
- Auto rides generated the highest revenue.
- UPI was the most frequently used payment method.
- Customer cancellations were lower than driver cancellations.
- Ride bookings remained relatively consistent throughout the year.
